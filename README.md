# Travel Creator Economy Jekyll Site

A Jekyll version of the Travel Creator Economy static site with blog support, topic/domain pages, reusable layouts, includes, and sample content.

## Included

- Jekyll `_config.yml`
- Reusable layouts for pages, posts, and topic pages
- Header and footer includes
- Blog index at `/blog/`
- Topic index at `/topics/`
- Topic collection pages for monetization, partnerships, audience growth, platform strategy, creator operations, and creator economics
- Three sample blog posts
- Original CSS and JavaScript moved into `assets/`
- `Gemfile`, `CNAME`, `.gitignore`, and MIT license

## Local development

```bash
bundle install
bundle exec jekyll serve
```

Then open `http://localhost:4000`.

## Configuration

Update `_config.yml` to change the site URL, description, and Substack URL.
